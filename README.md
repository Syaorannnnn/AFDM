# Affine Frequency Division Multiplexing Communication System Simulation

![MATLAB](https://img.shields.io/badge/MATLAB-R2023b%2B-blue.svg) 
本项目是一个基于 MATLAB 的 **仿射频分复用 (AFDM, Affine Frequency Division Multiplexing)** 通信系统链路级仿真平台。
AFDM 是一种针对高移动性、双色散（时变多径）信道设计的新型波形，能够通过调整 Chirp 参数获得全分集增益。
本工程采用了高度封装的面向对象设计 (`classdef`)，集成了从信道编码、调制、信道估计到高级均衡算法的完整物理层收发链路，特别侧重于在深衰落信道下的性能评估。

## 核心特性

* **面向对象架构**: 核心逻辑封装分别封装，参数配置集中、易于扩展和维护。
* **双色散信道模型 (LTV Channel)**: 支持自定义多径时延和多普勒频移的线性时变瑞利衰落信道。
* **信道估计**: 包含基于导频的信道估计，支持分数阶多普勒精搜索 (Fractional Doppler Search) 与动态加窗去噪。
* **高性能均衡算法**:
  * **MMSE (最小均方误差)**
  * **Weighted MRC-DFE (加权最大比合并判决反馈均衡器)**

## 环境依赖

* **MATLAB R2023b 或更高版本**
* **MATLAB Communications Toolbox**

# 更新日志

## [当前开发阶段] - 2026-03-06

### 新增
* **导频方案**: 引入 CAZAC 序列，替代传统的 ZP (Zero Padding) 填充单导频方案，增强系统的抗多径干扰能力。
* **信号处理**: 新增白化（Whitening）处理模块，通过消除噪声或干扰的特征相关性，为后续信道估计提供高可靠性数据输入。

### 变更
* **系统架构**: 重构核心架构，全面解耦各个功能组件，提升底层代码的可拓展性与后期维护的灵活性。

### 优化
* **信道估计流程**: 深度整合 CAZAC 序列与前置白化操作，提升了信道估计的精准度与鲁棒性。
* **误比特率**: 借助脉冲成型等新引入的信号处理技术，进一步改善了系统整体的 BER 性能指标。

## GI-Free (免 ZP 开销) 系统构建状态

**核心挑战：** 单导频子载波在信道估计起步阶段面临的严重 ID2P (Inter-Data-to-Pilot) 干扰。

### 核心迭代路径 (V1 - V5-A / 最终版)

为解决 ID2P 干扰问题，系统经历了整整五代优化，当前合并版本融合了以下关键技术节点：

* **V3 阶段 | 先验干扰正则化：** 保障了 OMP (正交匹配追踪) 算法在强干扰环境下的生存与可用性。
* **V4 阶段 | DD-CE (数据驱动信道估计)：** 利用数据估计结果，有效弥补了单导频带来的信息量缺失。
* **V5-A 阶段 | 截断均值正则化：** 实现了 LMMSE (线性最小均方误差) 正则化参数的优化，使其逼近理论最优状态。

### 性能评估与项目目标

| 维度 | 状态评估 | 备注说明 |
| :--- | :--- | :--- |
| **频谱效率** | 碾压态势 | 相比传统 EP 系统具有绝对优势 |
| **信道估计 (NMSE)** | 差强人意 | 高 SNR (信噪比) 区域仍有优化空间 |
| **误码率 (BER)** | 核心瓶颈 | 高 SNR 区域表现有待进一步突破 |

> **最终愿景：** 依托现有的极高频谱效率优势，进一步攻克高 SNR 区域的 BER 性能瓶颈，实现整个项目的圆满收官。
